# Política de Privacidad — Reels Automatizados IA

Última actualización: 2026-07-18

Reels Automatizados IA es un proyecto personal automatizado (GitHub Actions) que genera y publica videos cortos sobre curiosidades de inteligencia artificial.

## Datos que se procesan

El sistema solo procesa: texto generado por IA (guiones), imágenes generadas por IA, audio generado por texto a voz, y el video final resultante. No se recopila, almacena ni comparte información personal de terceros ni de usuarios externos.

## Uso de APIs de terceros

El proyecto usa las APIs oficiales de Google (Gemini, YouTube Data API), Hugging Face, Meta Graph API (Facebook/Instagram) y TikTok Content Posting API, únicamente para generar y publicar contenido en las cuentas propias del desarrollador. Las claves de acceso se almacenan como secretos cifrados en GitHub Actions y no se comparten con terceros.
